sequenceDiagram actor User participant LP as Login.tsx participant UA as useAuth Hook participant API as FastAPI /api/auth/login participant DB as employees table User->>LP: Enter email + password LP->>API: POST { email, password } API->>DB: SELECT * FROM employees WHERE email = ?
